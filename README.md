# 📖 About The Project

GramSahay AI is an intelligent multilingual platform designed to help rural citizens easily access government welfare schemes and digital assistance using Artificial Intelligence.

The platform simplifies:
- Government scheme discovery
- Eligibility guidance
- Rural digital accessibility
- Native language support
- AI-powered assistance
- Welfare awareness

GramSahay AI is specifically designed for:
- Farmers
- Rural families
- Students
- Women welfare beneficiaries
- Senior citizens
- Low digital-literacy users

# 🎯 Vision

To build an AI-powered digital ecosystem that empowers rural communities through accessible technology, multilingual communication, and intelligent welfare assistance.

# 🚀 Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/darsan-0/gramsahay-ai.git
```

---

## 2️⃣ Navigate to Project

```bash
cd gramsahay-ai
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Configure Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_APP_URL=http://localhost:3000
OPENAI_API_KEY=your_api_key
DATABASE_URL=your_database_url
```

---

## 5️⃣ Start Development Server

```bash
npm run dev
```

---

## 6️⃣ Open Browser

```bash
http://localhost:3000
```
